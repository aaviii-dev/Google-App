# Google Homepage Clone

A responsive Google Homepage Clone built using HTML and Tailwind CSS (CLI).

##  Features

- Google-inspired UI
- Responsive Layout
- Dark Theme Design
- Navigation Bar
- Search Bar Section
- Action Buttons
- Footer Section
- Built with Tailwind CSS CLI

##  Technologies Used

- HTML5
- Tailwind CSS
- Tailwind CLI

##  Project Structure

```bash
google-clone/
│
├── index.html
├── output.css
│
└── src/
    └── input.css
```

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/google-clone.git
```

Navigate to the project folder:

```bash
cd google-clone
```

Install Tailwind CSS:

```bash
npm install
```

Start Tailwind CLI:

```bash

```
